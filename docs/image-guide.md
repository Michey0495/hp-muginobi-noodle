# 画像差し替えガイド — 麦伸ぬーどる

本サイトではUnsplashのフリー素材をダミー画像として使用しています。
正式な写真素材をいただき次第、以下の箇所を差し替えてください。

- **サイズ**: 各640x640px以上推奨

## 画像の差し替え方法

1. 画像を `public/images/` に配置
2. 各コンポーネントの `src` プロパティを変更
   - 例: `src="/images/hero.jpg"`
3. `next.config.ts` の `remotePatterns` は不要になります（ローカル画像の場合）

## 注意事項

- 画像はWebP形式推奨（ファイルサイズ削減）
- `next/image` が自動最適化するため、元画像は高解像度で問題なし
- 食べログ等のユーザー投稿写真は使用不可
